Canvas API – Dibujando formas

Ejemplos basados en la documentación oficial de MDN

1) Rectángulos

Se utilizan los métodos fillRect, clearRect y strokeRect para dibujar, limpiar y contornear rectángulos directamente sobre el canvas.

2) Triángulo básico

Ejemplo de rutas (paths). Se inicia una ruta con beginPath, se define la forma con moveTo y lineTo, y se rellena usando fill.

3) Dos triángulos

Se dibujan dos triángulos: uno relleno y otro solo con contorno, demostrando el uso de fill, stroke y closePath.

4) Carita sonriente

Se usan arcos con arc para formar círculos y semicírculos, creando una cara con ojos y boca usando una sola ruta.

5) Matriz de arcos

Mediante ciclos for se dibuja una cuadrícula de arcos, alternando entre figuras rellenas y contorneadas, y variando los ángulos de dibujo.

6) Curvas Quadratic Bézier

Se emplea quadraticCurveTo para crear curvas suaves, formando un globo de diálogo a partir de varios segmentos curvos.

7) Curvas Cubic Bézier

Usando bezierCurveTo se construye una figura más compleja (un corazón), controlando la forma mediante puntos de control.

8) Combinación de formas

Ejemplo que mezcla rectángulos redondeados, arcos y curvas Bézier para construir una figura más elaborada a partir de formas básicas.

9) Path2D

Se crean objetos Path2D independientes para reutilizar rutas y dibujarlas con stroke y fill.

10) Path2D con SVG Path

Se utiliza un string con sintaxis SVG para crear una ruta mediante Path2D, demostrando la compatibilidad entre Canvas y SVG.
